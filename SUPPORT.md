## Common Questions

### Installation Issues

**Q: I'm getting compilation errors**
A: Ensure you have Rust 1.75+ installed. Run `rustup update` to update your toolchain.

**Q: Dependencies fail to build**
A: Some dependencies require system libraries. See the README for platform-specific prerequisites.

**Q: PostgreSQL connection errors (E² Multipass)**
A: Check your `.env` configuration and ensure PostgreSQL is running with the correct credentials.

### Usage Questions

**Q: How do I start a blockchain node?**
A: See the Quick Start section in [README.md](README.md) for step-by-step instructions.

**Q: Can I use this for cryptocurrency?**
A: No, BLS is designed for enterprise blockchain applications, not cryptocurrency trading.

**Q: What are the hardware requirements?**
A: See the README for recommended specifications. Requirements vary based on your use case.

### Development Questions

**Q: How do I contribute?**
A: See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed contribution guidelines.

**Q: Which branch should I target for PRs?**
A: Target the `main` branch unless otherwise specified by maintainers.

**Q: How do I run tests?**
A: Run `cargo test` for Rust tests and `npm test` for Node.js tests.
